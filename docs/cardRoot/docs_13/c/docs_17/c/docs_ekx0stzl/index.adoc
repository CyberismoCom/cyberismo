== Prerequisites

See {{#xref}}"cardKey":"docs_17"{{/xref}} for information about common prerequisites for all installation methods.

Install Node.js. The currently supported Node.js version is the latest LTS, v20.10.0 or later. Node.js is downloadable from https://nodejs.org/.

On macOS and Ubuntu/Debian, install the Clingo answer set programming runtime. This is not required on Windows, as Clingo is included in the Windows build of Cyberismo.

macOS (Homebrew): `brew install clingo`

Ubuntu/Debian: `sudo apt-get update && sudo apt-get install -y gringo`

== Installation

To install Cyberismo from the https://www.npmjs.com/package/@cyberismo/cli[public NPM registry], run the following command:

  $ npm install -g @cyberismo/cli
