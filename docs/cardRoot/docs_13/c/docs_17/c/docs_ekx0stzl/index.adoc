== Prerequisites

See {{#xref}}"cardKey":"docs_17"{{/xref}} for information about common prerequisites for all installation methods.

Install Node.js. Cyberismo requires Node.js v22 LTS or later. Node.js is downloadable from https://nodejs.org/.

The Clingo answer set programming runtime is bundled with Cyberismo as a prebuilt native binary, so no separate Clingo installation is required on Linux (x64, arm64), macOS (Intel, Apple Silicon), or Windows (x64).

== Installation

To install Cyberismo from the https://www.npmjs.com/package/@cyberismo/cli[public NPM registry], run the following command:

  $ npm install -g @cyberismo/cli
