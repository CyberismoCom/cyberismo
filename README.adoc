== Development

=== Environment
Copy *env.example* files to respective *.env* files

=== Instal Dependencies
[source, shell]
----
    pnpm install
----

=== Run Project
[source, shell]
----
    pnpm dev
----
Backend is accessible at http://localhost:3000

Frontend is accessible at http://localhost:5173
